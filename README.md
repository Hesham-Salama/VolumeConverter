# VolumeConverter
My first SwiftUI app ever.

iOS 13.3 - Xcode 10.3

It has a very simple UI which allows user to convert volumes. The included units: 
    
    * cup
    
    * fluid ounce
    
    * pint
    
    * quart
    
    * gallon
    
    * milliliter
    
    * liter
